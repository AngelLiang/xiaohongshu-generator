# 小红书文案生成器示例项目（v0.dev生成版本）

把v0.dev生成的代码下载到本地运行

v0.dev提示词

```
请帮我生成一个小红书文案生成器的网站，需要有首页和主要功能等。

## 技术栈要求
- 使用nextjs，需要使用nextjs的app router模式
- 使用tailwindcss
- 使用javascript，不要使用typescript
```

v0.dev分享链接：https://v0.dev/chat/next-js-text-generator-2WTEZ79KB7n?b=b_8Vkp104wgv0&p=0

## 快速开始

安装依赖

```
npm install
```

启动服务

```
npm run dev
```
